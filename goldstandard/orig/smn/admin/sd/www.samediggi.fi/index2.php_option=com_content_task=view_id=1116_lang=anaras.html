Sämitige škovlim- já oppâmateriaaltoimâttâh uásálist ive 2015 Turku almugijkoskâsâš kirjemessuid, kost tot čaaitâš sämikielâlii oppâmateriaal, sämimáttááttâs, sehe Säminuorâi taaiđâtábáhtus.

Paaneelsavâstâlmist suogârdillii já savâstillii sämikielâi puátteevuođâst.
