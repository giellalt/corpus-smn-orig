Aanaar markkân tuáimeeh láá ovtâstittâm vuoimijdis já ráhtám ohtsii ohjelm säämi aalmugpeeivi kunnen. Aalmugpeeivi ohjelmist lii puohtum oovdân säämi kulttuur maaŋgâpiälásávt. Láá muusik, elleekoveh, kovetaiđuuh, mainâstem, poccuuh, irâttâsvuotâ já njunošartisteh! Meid puoh Suomâst sarnum sämikielah láá valdum vuotân. Ohjelm pištá iđedist ehidân já tot lii uáivildum jyehi ahasijd! Tiervâpuáttim navdâšiđ peeivi ohjelmfalâlduvvâst já ilolii oovtâst oromist! 

Siijdâst láá ávus uuvsah tme 9-17. Siijdâ šiljoost lii aalmugpeeivi lekkâm sehe liipu pajedem. Sämimuseo lekkâs uđđâ ohtsâščáitálduv, Säminuorâi taaiđâtábáhtus uápisin, peeivi aalgân. Säämi máttááttâskuávdáá uáppeeh láá ráhtám vijđes já maaŋgâpiälásii kulttuurohjelm Siijdân.
 
Kulttuurkuávdáš Sajosist álgá meid kulttuurohjelm jo iđedist. Sajosist lii meid Säämi máttááttâskuávdáá pyevtittem ohjelm. Meid Aanaar peivikiäčču lii puáhtám jieijâs ohjelm piäiván. Sajosist láá ávus uuvsah já táálu tuáimeeh oovdânpyehtih jieijâs.
 
Sämitige aalmugpeeivi juhle álgá Sajosist tme 15.30 rääjist paarnij puásuivyejimijguin. Aalmugpeivikaahvijd fäällih tme 16.00 já aalmugpeeivi juhle lii tme 17.00-19.00. Liägus kulttuur tiäduttettee ehidân láá puohah tiervâpuáttim!
 
Aalmugpiäiván láá Aanaar markkânist kuullâm kuhháá meid Aanaar servikode já SámiSoster ornim purâdem servikoddetáálust sehe kirhomiänuh. Ärbivyehi juátkoo meid taan ive. Aalmugpeeivi pispemessu lii Aanaar kirhoost tme 15.00, siärvástâllâm ja oovtâstorroom purâdem paaldâst algâtteh jo koskâpeeivi ääigi.
 
Aanaar tábáhtusijd čuávu ubâ peeivi Yle Säämi, mon ohtsâštave-eennâmliih vuolgâttâsah älgih aarriiđeed tme 7.10. Eromâš mainâšummeen Andte Gaup-Juuso, kote lii Yle Oddasat –vuolgâttâs uđđâsij lohhen tme 15.10 Yle TV1:st!
 
Tiervâpuáttim Anarân!
