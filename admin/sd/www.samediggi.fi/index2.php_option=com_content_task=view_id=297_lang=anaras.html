Säminuoráid jieijâs orgaan sämitige ohtâvuotân
Säminuoráid vuávájuvvoo sierânâs orgaan sämitiigán. Nuorâiorgaan vuávám lii lamaš joođoost máttááttâsministeriö ruttâdem projektin ive aalgâ rääjist.

 

Sämitigge lii asâttâm ääši valmâštâllâm várás sierânâs pargojuávhu, mon saavâjođetteijen tuáimá sämitige nuorâmus jeessân Tuomas Aslak Juuso já mon áášij valmâštellen parga proojeektpargee Anne Länsman. Máttááttâsministeriö lii mieđettâm sämitiigán 40.000 euro lasetorjuu vuávámpargo juátkimân. Sämitigge páhudij tuđâvâžžân čuákkimistis nuorâi ääši ovdánem.

Viimeksi päivitetty ( 24.06.2009 )
