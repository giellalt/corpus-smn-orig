Nelji uđđâ tavesämikielâlâš matematiik kirje almostuvvii
Vuáđumáttááttâs uđđâ matematiik oppâkirjerááiđun láá almostum nelji uđđâ tavesämikielâlâš oppâkirje. Uđđâ oppâkirjeh láá kiđđâluuhâmpaje uáppeekirjeh 1–4 luokkáid. Lohkoleaika- nommâsiih oppâkirjeh láá jurgâlum WSOY Matikka-nommâsâš rááiđust. Tavesämikielâliih čohčâluuhâmpaje oppâkirjeh 1–4 já aanaarsämikielâlâš čohčâluuhâmpaje oppâkirje Lohosierâ 1 lii almostittum ive 2011. Ubâ ráiđu almostittoo aanaar- já tavesämikielân.

 

Oppâkiirjijd puáhtá tiiláđ sämitige škovlim- já oppâmateriaaltoimâttuvâst: anni.nakkalajarvi(at)samediggi.fi tâi ritva.tammela(at)samediggi.fi.

 

Lasetiäđih täst.
