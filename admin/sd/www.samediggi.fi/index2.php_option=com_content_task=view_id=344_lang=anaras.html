Toimâstivrejeijee
Sämitige ohtâvuođâst tuáimá Säämi párnáikulttuurkuávdáš, mii lii uassin Laapi párnáikulttuurkuávdášviärmáduv já tađe mield meiddei väldikodálâš Tijdâlamppu –viärmáduv jeessân. Ko toimâstivrejeijee páácá virgerijjâvuotân almoot Sämitigge uuccâmnáál toimâstivrejeijee sajasâšvuođâ 20.1. – 29.11.2010 koskâsâš ááigán.
