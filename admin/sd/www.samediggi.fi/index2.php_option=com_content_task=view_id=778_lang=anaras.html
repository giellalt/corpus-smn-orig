Sämitige ive 2013 kielâtaho -palhešume Yle Sápmin

Sämitigge lii mieđettâm Säämi kielâtaho- palhâšume Yle Sápmin. Palhâšume mieđettuvvoo jyehi nubbe ive sämikielâ sajattuv já sämikielâlij palvâlusâi oovdedmân porgum ánsulâš pargoost.

 

Säämi kielâtaho -palhâšume 2013 lii Outi Pieski kyehti maalâkove ”Nuvvos Áiligas, eennâm suddá” já ”Kolle-eennâm”.

Viimeksi päivitetty ( 19.12.2013 )
