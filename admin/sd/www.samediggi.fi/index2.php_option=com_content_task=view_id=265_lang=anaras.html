Ij‚ttis Ijj‚ muusikfestivaal‚ porgem··nust 2009
Algâaalmugij muusikfestivaalâ Ijâttis Ijjâ uárnejuvvoo puáttee ive porgemáánu 21.-23. peeivij.

 

Festivaalâ lii uárnejum ive 2004 rääjist já ovdebijn iivijn vyesimáánu loopâst. Festivaalâ ohjelmvuávám piso siämmáálágánin. Ohjelmist puátá leđe mieldi sehe säämi já eres algâaalmugij muusik. Lasetiäđuh: www.ijahisidja.fi .

Viimeksi päivitetty ( 03.12.2008 )
