Sämitige ornim Helsig seminaar fáddán lii sämikielâ já maorikielâ iäláskittem já nanodem.  Seminaar ohtân uáivilin lii tieđettiđ sämitige uulmijn já tooimâin siäiluttiđ já ovdediđ sämikielâ sehe paijeentoollâđ ton eellimvuáimálâšvuođâ.
