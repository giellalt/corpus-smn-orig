SaKaste lii sämmilij sosiaal- já tiervâsvuođâpalvâlusâi ovdedemhaahâ, mii vuálgá joton sämikuávlust. Tot lii uási Tave-Suomâ maaŋgâsuárgásiih sosiaal- já tiervâsvuođâpalvâlusah (PaKaste 2) –haavâ, mon ruttâd STM haldâšem Sosiaal- já tiervâsvuođâhuolâttâs aalmuglâš ovdedemohjelm (KASTE).

 

SaKaste -haavâ ulmen lii pyerediđ sämmilij jieijâs kielân já kulttuurân vuáđuduvvee palvâlusâi finnim, kvaliteet sehe vaikuttemvuođâ, oovdedmáin sämmilij vuáđuvuoigâdvuođâi já oovtviärdásâšvuođâ olášume sehe uásálâšvuođâ. Sämmilij táárbuin vyelgee palvâlusah ovdedeh sämmilâš ohtâgâsâi já siärvusij tiervâsvuođâ sehe pyereestvaijeem, kepideh helbâm já kezideh tiervâsvuođâiäruid.

Viimeksi päivitetty ( 21.10.2011 )
