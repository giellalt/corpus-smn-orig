Sämitige Anarist 15.2. toollum tievâsčuákkim nomâttij lävdikudijdis já ovdâsteijeidis oovtâstpargo-orgaanáid vaaljâpajan 2012–2015. Škovlim- já oppâmateriaallävdikode joođeet Yrjö Musta, kulttuurlävdikode Tauno Haltta, säämi kielârääđi Anna Morottaja, iäláttâs- já vuoigâdvuođâlävdikode Nilla Tapiola, sosiaal- já tiervâsvuođâlävdikode Arto Pohjanrinne já nuorâirääđi Laura-Maija Niittyvuopio.
