Sämitigge ovdeed päikkisiijđoidis
Sämitigge lii algâttâm päikkisiijđoidis uđâsmittem. Päikkisijđoid lii lasettum uđđâ tiätu sämmilijn lahâaasâtmist , sämitige ovdâsteijein jieškote-uv pargojuávhuin , algâaalmugij vuoigâdvuođâid kieđâvuššee aalmugijkoskâsijn sopâmušâin  sehe algâaalmugij vuoigâdvuođâi olášume koččee orgaanij uđđâsumos avžuuttâsâin  Suomâ staatân. Uđđâ siijđoh jurgâluvvojeh eidu sämikielân.
Viimeksi päivitetty ( 15.01.2014 )
