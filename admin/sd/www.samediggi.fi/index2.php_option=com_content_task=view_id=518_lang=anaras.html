Sämmilij kielâliih vuoigâdvuođah iä olášuu
Sämitige värisaavâjođetteijee Irja Seurujärvi-Kari tiäduttij sänivuárustis kielâiäláskittemseminaarist 13.9.2011 tom, et sämmilij kielâliih vuoigâdvuođah iä olášuu Suomâst vuáđulaavâ uáivildem vuovvijn.

 

Njuálguliŋkkâ I värisaavâjođetteijee Irja Seurujärvi-Kari sänivuárun 13.9.2011 tast.

Viimeksi päivitetty ( 23.09.2011 )
