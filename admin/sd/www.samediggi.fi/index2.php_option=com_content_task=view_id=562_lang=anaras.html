Tiina Sanila-Aikio já Heikki Paltto sämitige värisaavâjođetteijen
Čevetjäävrist šaddaaš nuorttâsämmilâš Tiina Sanila-Aikio já Heikki Paltto Aanaar Lemmest väljejuin oovtmielâlávt sämitige värisaavâjođetteijen vaaljâpajan 2012–2015. Sanila-Aikio, kote väljejui vuossâmužžân värisaavâjođetteijen, lii sämitiggeovdâsteijen vuossâmu paajees. Paltto, kote väljejui nubben värisaavâjođetteijen, čokkái sämitige stiivrâst jo ovdebáá vaaljâpaajeest.

 

Sämitige stiivrân väljejuvvojii jienâstem maŋa čuávvoo jesâneh:

Ucjuuvâst Ilmari Tapiola já Asko Länsman

Anarist/ Helsigist Pirita Näkkäläjärvi

Suáđigil Vuáčust Petra Magga-Vars.

Viimeksi päivitetty ( 15.02.2012 )
