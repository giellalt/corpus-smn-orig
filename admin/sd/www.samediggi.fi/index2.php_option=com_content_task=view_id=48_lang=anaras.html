Kulttuurlävdikodde

Kulttuurlävdikodde lii sämitige stiivrâ vuálásâš toimâorgaan, mii valmâštâl sämitige iävtuttâsâid, alguid sehe ciälkkámušâid, moh kyeskih sämmilij jiegâlâš kulttuur. Kulttuurlävdikode pargon lii meridiđ staatâtorjuu (säämi kulttuurmeriruttâ) jyehimist sehe čuávvuđ já kocceeđ mieđettum torjui kevttim. Lävdikodde kalga rähtiđ ihásávt sämitige čuákkimân čielgiittâs säämi kulttuurmeriruuđâ kevttimist já toohâđ iävtuttâs staatâ budjetân ihásâš säämi kulttuurmeriruuđâst.

 

Kulttuurlävdikode aašij valmâštellen já oovdânpyehten tuáimá kulttuurčällee.
