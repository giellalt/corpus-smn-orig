Tavesämikielâlâš aabbis nubbe pargokirje almostui
Tavesämikielâlâš aabbis ´Áppes - Min meahci máidnasat´ kiđđâpaje pargokirje lii almostum. Pargokirjeest láá puohnâssân 126 sijđod já tuálá siste hitruus já ivnáás eenikielâ tááiđu ovdedeijee pargopittáid.

 

Kirje teevstah já pargopitáh vuáđuduveh aabisrááiđu čällee Inga Guttorm ovdil almostum siämmáánommâsâš aabbis tevstáid. Kirje kuvvim lii toohâm Salli Parikka Wahlberg. Kirje lii finniimist sämitige škovlim- já oppâmateriaaltoimâttuvâst Sajosist Anarist. Sämitige almostittem oppâmateriaaleh kávnojeh täst.

Viimeksi päivitetty ( 13.08.2012 )
