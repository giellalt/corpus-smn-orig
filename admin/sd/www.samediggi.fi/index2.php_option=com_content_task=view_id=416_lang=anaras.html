Šiljopiirâs

Kulttuurkuávdáá šiljopiirâs heivee ereslágán olgotábáhtussáid, ovdâmerkkân messutilálâšvuođâi olgouásáduvâi já muusikfestivaalij táárboid.

 

 

Säämi kulttuurkuávdáš Sajos lii Juvduujuv riddoost, kaskoo Aanaar markkân. Aanaarjáávrán lii mätki suulân 200 meetterid. Šiljoost lii valjeest parkkisaje čuákkim- ja tábáhtusäššigâssáid.

 

Saijoos raavâdviäsu Galla palvâl olgotábáhtusâi ääigi festivaalraavâdviässun. Raavâdviäsust láá oles juvâstâhvyebdimvuoigâdvuođah.
