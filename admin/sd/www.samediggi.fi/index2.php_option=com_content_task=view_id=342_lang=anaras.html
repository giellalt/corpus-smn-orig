Vuossâmuš tavesämikielâlâš škoovlâ kárttákirje almostui
Vuossâmuš tavesämikielâlâš kárttáčuágáldâh škoovlâi várás lii almostum. ´Skuvlla kártagirji´ lii uáppee vuáđučuágáldâh já tast kávnojeh tiäđuh Suomâst, Tave-enâmijn já sierâsnâs eennâmoosijn. Káártái lasseen kirjeest lii luvâttâllâm jiečânâs staatâin oovtâst jieijâs lipuiguin já tavesämikielâlâš usâttâh. Luuvâ lase täst.
Viimeksi päivitetty ( 12.11.2009 )
