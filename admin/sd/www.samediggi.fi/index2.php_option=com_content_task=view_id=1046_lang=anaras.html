Pargo álgá 9.6 já pištá 28.8.2015 räi. Proojeektpargee pargoid kuleh el. säämi muusikkuávdáá išedeijen toimâm sehe Ijâttes ijjâ- algâaalmugij muusikfestivaal išepyevtitteijee pargoh.

 

Pargo miänástuvvee hoittám išedeh hiäivulâš škovlim, sämikielâ táiđu sehe kelijdeijee eŋgâlâskielâ táiđu. Proojeektpargee pälkki miärášuvá Suomâ sämitige pálkkááttemvuáháduv vátávâšvuođâtääsi VII/I mield (vuáđupälkki 1711,31eurod/mp). Vuáđupäälhi lasseen máksojeh 24 % sämikuávlu lase já pargohárjánem mield miärášuvvee hárjánemlaseh. Proojeektpargee toimâsaje lii säämi kulttuurkuávdáš Sajosist.
