Ovdâskoddevaaljah tuállojeh jyehi niäljád ive
Vaaljâin väljejuvvojeh 200 aalmugovdâsteijed meridiđ mii ohtsijn aašijn. Ive 2011 ovdâskoddevaaljah uárnejuvvojeh pasepeeiivi 17.4.2011. Lasetiäđuh vaalit.fi -siijđost paječálusist “Tietoa eduskuntavaaleista eri kielillä” tâi taam njuálguliiŋkâst (PDF-tiätuvuárkká). 
Viimeksi päivitetty ( 24.03.2011 )
