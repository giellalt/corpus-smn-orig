Sajosist uárnejuvvojeh jo kuálmád keesi ääigi sämituojij pargopáájáh. Kietâtyejipáájá stivrejeijee paargon kuleh rađđiđ äššigâsâid sämituojij tuoijumist já mainâstiđ sämituojij historjást já ärbivuovijn sehe jieijâs kietâtyejivuovijn. Kietâtyejipáájá stivrejeijee pargosaje lii Säämi kulttuurkuávdáš Sajosist Anarist.

 

Pargo álgá 15.6.2015 já pištá 14.8.2015 räi. Pargo miänástuvvee hoittám išedeh hiäivulâš škovlim, sämikielâ sehe eŋgâlâskielâ táiđu. Hiättun lohhoo eres kielâi táiđu. Pälkki miärášuvá Sämitige pálkkááttemvuáháduv vátávâšvuođâtääsi VII/III miäldásávt (vuáđupälkki 1545,10 eurod/mp). Vuáđupäälhi lasseen máksojeh 24 % sämikuávlu lase já pargohárjánem mield miärášuvvee hárjánemlaseh.
