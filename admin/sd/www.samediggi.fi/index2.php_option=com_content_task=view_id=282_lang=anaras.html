Aanaar paješkoovlâ teatterjuávkku Beaivvi áhkubat vuálgá ovdâstiđ säminuorâid väldikodálâš Nuorâ Kulttuur -tábáhtusân Lappeenranta-kaavpugân vyesimáánust. Tábáhtus tuámárin toimáin caittâleijee-toimâtteijee Irene Länsman já teatteripargo uáppee Heli Huovinen.
