Alguid sämitige noomâ rievdâdmist Säämi Parlamenttân já saavâjođetteijee pargonomâttâs rievdâdmist presidenttân vuolgâttuvvojeh täärhib valmâštâlmân.

´Säämi Parlament´-nomâttâs kuvviiččij Länsmanin mield eenâb sämitige pargoid já sajattuv. Sämitige saavâjođetteijee pargonomâttâs rievdâdmist ´presidenttân´ lasettičij sämitige áárvustan-neem já kuvviiččij saavâjođetteijee tuođâlâš sajattuv aalmug ovdâsteijee orgaan saavâjođetteijen.
