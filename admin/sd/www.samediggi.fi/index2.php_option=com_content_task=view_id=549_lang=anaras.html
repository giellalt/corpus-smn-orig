Ive vuossâmuuh uđđâ oppâmateriaaleh almostuvvii
Ive 2012 vuossâmuuh sämikielâliih oppâmateriaaleh láá almostum. Vuáđumáttááttâs 9. luoka ohtsâškoddetiäđu tavesämikielâlâš táiđuvihko Servodat dál 9, dáidogihpa lohtâs siämmáánommâsii ovdeláá almostittum oppâkiirján. Luvâttuv oppâkirje Giela geainnut 1 meerhâš aalgâ vuossâmuu tavesämikielâlâš eenikielâ oppâkirjerááiđun.

 

Uđđâ já ovdeláá almostittum sämikielâliih oppâmateriaaleh láá finniimist sämitige škovlim- já oppâmateriaaltoimâttuvâst säämi kulttuurkuávdáš Sajosist Anarist. Lasetiäđuh täst.

 

Luvâttâllâm ihán 2011 mieđettum budjetruuđâin almostittum oppâmateriaalijn kávnoo täst.

Viimeksi päivitetty ( 18.01.2012 )
