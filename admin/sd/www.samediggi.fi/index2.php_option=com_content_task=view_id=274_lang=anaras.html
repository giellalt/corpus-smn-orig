Almottâttâm Säminuorâi taaidâtábátusân lii álgám
Almottâttâm Säminuorâi taaidâtábátusân, mii uárnejuvvoo Vuáčust njuhčâmáánu 26. peeivi, lii álgám. Tábáhtus teeman lii teatter já uánihiselleekove. Almottâttâm teatterkištodmân nohá 2.3. já elleekovefestivaalân 13.3. Tärhibeh tiäđuh almottâtmist täst: Taidetapahtuman 2009 säännöt - inarinsaame já Almottâtmeh_2009.
Viimeksi päivitetty ( 19.02.2009 )
