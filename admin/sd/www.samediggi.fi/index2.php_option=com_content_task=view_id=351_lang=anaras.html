Säminuorâi taaidâtábáhtus Ruávinjaargâst 7.-8.4.2010
Säminuorâi taaidâtábáhtus uárnejuvvoo kyevtipiäivásâš tábáhtussân Ruávinjaargâst nuorâitáálu Mondest kuávdoo kaavpuh cuáŋuimáánu 7.–8. peeivi. Väldikodálâš Nuorâ Kulttuur –tábáhtus ij uárnejuu taan ive ollágin, et Säminuorâi taaidâtábáhtus lii aaibâs jieijâs jiečânâs tábáhtus. Tábáhtus teeman lii muusik. Taaidâtábáhtustiäđáttâs 27.1.2010.
Viimeksi päivitetty ( 27.01.2010 )
