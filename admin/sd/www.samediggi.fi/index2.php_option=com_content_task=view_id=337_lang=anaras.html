Säminuorâi taaidâtábáhtus 2010 Ruávinjaargâst
Säminuorâi taaidâtábáhtus 2010 uárnejuvvoo Ruávinjaargâst njuhčâmáánust kyevtipiäivásâš tábáhtussân. Sämitige škovlim- já oppâmateriaaltoimâttâh lii porgâškyetmin taaidâtábáhtus orniimijn ovtâspargoost Ruávinjaargâ kaavpugáin. Puátimin lii ivnáás säminuorâi kulttuur- já taaidâjuhle! Čuávu nettiuđđâsijd www.samediggi.fi .
Viimeksi päivitetty ( 27.10.2009 )
