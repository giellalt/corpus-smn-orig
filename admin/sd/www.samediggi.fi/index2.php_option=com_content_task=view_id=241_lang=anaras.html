Kárvudât peri návt

Keejâ uánihisvideo sämimáccuhân kárvudâtmist.

Ingá, Ánne já Niiles peessih riipân já kárvudeteh sämimáccuháid. Kárvudâttâm ij lah kuittâg meendu älkkee, pic tärhis pargo, moos taarbâš iše.
