Sämikulttuurkuávdáš

Säämi kulttuurkuávdáš, mii huksejuvvoo Aanaar kirkkosiidjân Juvduu riidon, šadda leđe Suomâ sämmilij haldâšem já kulttuur kuávdáš. Säämi kulttuurkuávdáá uáivilin lii orniđ sämmiláid pyerebijd iävtuid alnetoollâđ já ovdediđ jiešráđálávt kielâs, kulttuuris já iäláttâstooimâs sehe hoittáđ já ovdediđ kulttuurjiešhaldâšemes já tuárjuđ sämmilij almos eellimtile ovdánem. Kulttuurkuávdáš pyereed meiddei merhâšitteht máhđulâšvuođâid levâttiđ tiäđu sämmilijn algâaalmugin. Säämi kulttuurkuávdáš šadda leđe aalmuglávt já aalmugijkoskâsávt-uv mielâkiddiivâš já pivnohis maaŋgâkulttuurlâšvuođâ kuáhtámpäikki.

 

Sämikulttuurkuávdáá ruttâdmân lii finnejum 5 miljovn eurod Euroop union kuávluovdedemruttârááju vaarijd.
