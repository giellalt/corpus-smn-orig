Sämitigge ocá raavâdviäsuirâtteijee Sajosân
Säämi kulttuurkuávdáš Sajos lii kuávdášlâš saajeest Aanaar markkânist Juvduujuv riddoost. Sajos tuáimá Suomâ sämmilij parlamenttáállun, kulttuur-, škovlim- já mättimkuávdážin sehe tábáhtustáállun.

 

60-sajasâš raavâdviäsu lii tergâdis uási čuákkim-, kongres- já kulttuurtábáhtusâi ornim. Saijoos toimâ lii aalgâst, nuuvt ete tooimâ innovatiivlâš oovdedmân láá šiev máhđulâšvuođah.

 

Tuáivup, ete raavâdviäsuirâtteijee piäjá tooimâ joton 1.5.2014 rääjist teikâ sopâmuš mield.

 

Raavâdviäsutooimâst perustum ulmuid pivdep väldiđ ohtâvuođâ 15.4.2014 räi.
