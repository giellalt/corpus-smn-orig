Algâaalmugijd kyeskee julgáštus
Olgoašijministeriö lii jurgâlittám OA julgáštus suomâkielân, mii kuáská algâaalmugij vuoigâdvuođâid já sämikielâlâš jurgâlus lii pargo vyelni. Ive 2007 tuhhiittum algâaalmugijd kyeskee julgáštus (United Nations Declaration on the Rights of Indigenous Peoples) lii Suomâ staatâ kenigittee äššikirje. Algâaalmugijd kyeskee julgáštus lii poolitlâš äššikirje já Suomâ lâi aktiivlávt tuárjumin julgáštus tuhhiittem OA:st.
Viimeksi päivitetty ( 15.02.2011 )
