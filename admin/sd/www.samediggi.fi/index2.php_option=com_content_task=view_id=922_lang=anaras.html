Oppâmateriaalpaargon kuleh el. kietâčallui čäällim, jurgâlem, kielâ tärhistem, kuvvimpargo, tuáijim, ohjelmistem jna. Tilálâšvuođâst oovdânpuáhtoo sämitige oppâmateriaalpargo, suunnâttâllâp já savâstâllâp. Kustan-nusosakeyhtiö Otava oppâmateriaalpyevtittâsâst muštâl kuástidemhovdâ Mervi Korhonen Helsigist. Lasetiäiđuid tábáhtumest addel Hannu Kangasniemi (hannu.kangasniemi(at)samediggi.fi/ 050-3641 385).
